# Desafio Prático Front-End - 

## Um app simple de cadastro

Objetivo é que o usuário consiga de uma maneira simples gerar uma tabela de dadso através de um formulário.

## Ferramentas:

- HTML
- JS
- Jquery
- Bootstrap

### Decisões adotadas 

Partindo da idéia de que qualquer usuário consiga usar de uma forma simple e gerar uma tabela de dados sem qualquer conhecimento de sistema.
